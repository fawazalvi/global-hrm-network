# **App Name**: TalentVerse

## Core Features:

- Secure User Authentication: Firebase Authentication with email/password and Google OAuth. Mandatory email verification and password reset flow.
- Role-Based Access Control: Defines roles (user, admin, moderator) with specific permissions to ensure data security and platform management.
- HR Profile Management: Allows HR professionals to create, edit, and manage their profiles, including experience, skills, certifications, and achievements.
- Profile Verification Workflow: Admin-moderated process to verify user profiles, ensuring the authenticity and credibility of the platform's members.
- Subscription Management: Handles user subscriptions (free/pro/enterprise) with status updates and integration with Stripe for payments.
- Profile Score Calculation Tool: Generative AI algorithm that evaluates HR profiles based on completeness, relevance, and other factors to generate a profile score, helping users to know when to display the profile or keep it unlisted. The AI LLM will be used as a tool.
- Admin Panel: Provides a dashboard for user management, profile verification, content moderation, and subscription overview.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), for trust, professionalism, and innovation.
- Background color: Very light cyan (#E0FFFF), providing a clean and calming backdrop that helps content stand out.
- Accent color: Violet (#8A2BE2), introducing a touch of creativity and sophistication.
- Body and headline font: 'Inter', a sans-serif typeface offering excellent readability and a modern aesthetic.
- Use clean and professional icons that represent HR functions and profile elements.
- A clean and intuitive layout with clear navigation to facilitate user engagement.
- Subtle transitions and animations to enhance user experience without being distracting.
